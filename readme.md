## Test Automation with JUnit ##

### Stack Exercise ###

Übungsprojekt welche uns das automatische Testen mittels JUnit näher bringen soll.

Es wurden dem vorgegeben Projekt verschiedene Test hinzugefügt sowie z.B. die Überprüfung
ob die Funktionen push und pop richtig funktionieren.

Hierfür wurden die assertTrue, assertFalse Methoden welche vom JUnit Framework zur verfügung 
gestellt werden, verwendet. 

Desweiteren wurden noch das Exceptionhandling überprüft.

In der Testclasse wurde eine Setup mit @Before hinzugefügt wo vor jedem Test ein Stack angelegt wird mit der Größe 5.
